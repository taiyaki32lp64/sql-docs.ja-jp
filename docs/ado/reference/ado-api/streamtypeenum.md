# <a name="streamtypeenum"></a>StreamTypeEnum
格納されたデータの種類を指定します、 [Stream](../../../ado/reference/ado-api/stream-object-ado.md)オブジェクト。  
  
|定数|Value|説明|  
|--------------|-----------|-----------------|  
|**adTypeBinary**|1|バイナリ データを示します。|  
|**adTypeText**|2|既定値です。 文字で指定されたセット内にあるテキスト データを示す[Charset](../../../ado/reference/ado-api/charset-property-ado.md)します。|  
  
## <a name="adowfc-equivalent"></a>ADO と WFC と同等  
 これらの定数には、ADO と WFC 対応はありません。
