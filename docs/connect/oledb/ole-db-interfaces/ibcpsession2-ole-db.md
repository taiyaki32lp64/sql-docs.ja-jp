# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  IBCPSession2 インターフェイスは、IBCPSession の拡張機能であり、各列に対して IBCPSession:: BCPColFmt を呼び出す代わりに、メンバー関数を提供します。  IBCPSession2 は IBCPSession から継承し、新しいメソッドを[IBCPSession2:: BCPSetBulkMode](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md)1 つ追加します。
