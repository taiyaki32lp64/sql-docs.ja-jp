# <a name="icolumnsrowset"></a>IColumnsRowset
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  OLE DB Driver for SQL Server は、DBCOLUMN_BASETABLEINSTANCE 列を IColumnsRowset:: GetColumnRowset に追加します。 この列は DBTYPE_I2 を返します。この列は、マイクロソフトが使用するために予約されています。 この列の情報は今後のリリースで変更されます。
