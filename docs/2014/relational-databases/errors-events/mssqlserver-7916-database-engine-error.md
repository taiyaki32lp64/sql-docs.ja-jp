# <a name="mssqlserver7916"></a>MSSQLSERVER_7916
    
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|SQL Server|  
|イベント ID|7916|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名|DBCC2_REPAIR_RECORD_DELETED|  
|メッセージ テキスト|修復:オブジェクト ID O_ID、インデックス ID I_ID、パーティション ID PN_ID、アロケーション ユニット ID A_ID (型 TYPE)、ページ P_ID、スロット S_ID のレコードを削除します。 インデックスが再構築されます。|  
  
## <a name="explanation"></a>説明  
 これは、REPAIR からの情報メッセージであり、このページから指定のレコードが削除されたことを示しています。
