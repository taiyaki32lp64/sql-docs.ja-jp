# <a name="kpiweight-mdx"></a>KPIWeight (MDX)


  指定された主要業績評価指標 (KPI) の重みを返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## <a name="arguments"></a>引数  
 *Kpi 名*  
 KPI の名前を指定する有効な文字列式。  
  
## <a name="remarks"></a>コメント  
 返される値は、親 KPI の比重です。
