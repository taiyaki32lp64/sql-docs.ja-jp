# <a name="using-mathematical-functions"></a>数学関数の使用


  数学関数は数値式の数学演算を実行し、演算の結果を返します。  
  
 特に指定しない限り、数学関数に引き渡される数字は、倍精度浮動小数点数として解釈されます。
