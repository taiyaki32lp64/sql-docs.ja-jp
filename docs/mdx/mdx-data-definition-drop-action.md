# <a name="mdx-data-definition---drop-action"></a>MDX データ操作 - DROP ACTION


  指定されたキューブから、指定したアクションを削除します。  
  
## <a name="syntax"></a>構文  
  
```  
  
DROP ACTION CURRENTCUBE | Cube_Name  
   .Action_Name   
```  
  
## <a name="arguments"></a>引数  
 *Cube_Name*  
 キューブ名を提供する有効な文字列式。  
  
 *Action_Name*  
 削除するアクションの名前を指定する有効な文字列式です。
