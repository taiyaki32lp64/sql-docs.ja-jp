# <a name="sinstance-class"></a>SInstance クラス
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [Sinstance クラス](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md)は [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]のインスタンスを表します。  
  
 [Sinstance クラス](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md)は、[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]のインスタンスに関連付けられたプロトコルとネットワークライブラリを管理するために使用できます。 また、セキュリティ証明書を [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] のインスタンスに対して指定することもできます。
