# <a name="msmerge_current_partition_mappings"></a>MSmerge_current_partition_mappings
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  **MSmerge_current_partition_mappings**テーブルが変更された行が属する各パーティションの id の 1 つの行を格納します。 このテーブルは、パブリケーション データベース内に保存されます。  
  
|列名|データ型|説明|  
|-----------------|---------------|-----------------|  
|**publication_number**|**smallint**|格納されているパブリケーションの番号**sysmergepublications**します。|  
|**tablenick**|**int**|パブリッシュされたテーブルのニックネームです。|  
|**rowguid**|**uniqueidentifier**|特定の行の行識別子。|  
|**partition_id**|**int**|行が属しているパーティションの ID。 行の変更がすべてのサブスクライバーに関連する場合、値は-1 を使用します。|
