# <a name="data-collector-views-transact-sql"></a>データ コレクターのビュー (Transact-SQL)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  データ コレクターは、コレクションの設定時に取得される実行の統計に加え、コレクター型のプロパティ、コレクション セット、およびコレクション セット アイテムなどのデータ コレクターの構成に関する情報を表示するために、次のビューを提供しています実行します。 これらのビューでは、 **msdb**データベースでも、基になるテーブルの抽象化レイヤーを提供します。 この抽象化では、関連するアプリケーションに影響を与えずに、テーブルへの変更を許可する一方、テーブルに直接アクセスを防止することでセキュリティを強化します。
