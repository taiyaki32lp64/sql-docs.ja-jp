# <a name="using-scrollable-cursors"></a>スクロール可能なカーソルの使用
スクロール可能なカーソルを使用するには、これら 3 つの手順が必要です。  
  
1.  カーソル機能を決定します。  
  
2.  カーソルを設定します。  
  
3.  行のスクロールとフェッチします。  
  
 このセクションでは、次のトピックを扱います。  
  
-   [カーソル機能の決定](../../../odbc/reference/develop-app/determining-cursor-capabilities.md)  
  
-   [カーソルの設定](../../../odbc/reference/develop-app/setting-up-the-cursor.md)  
  
-   [カーソルの特定とカーソルの種類](../../../odbc/reference/develop-app/cursor-characteristics-and-cursor-type.md)  
  
-   [行のスクロールとフェッチ](../../../odbc/reference/develop-app/scrolling-and-fetching-rows-odbc.md)
