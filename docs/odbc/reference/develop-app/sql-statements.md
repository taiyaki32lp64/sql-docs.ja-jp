# <a name="sql-statements"></a>SQL ステートメント
ODBC アプリケーションでは、SQL ステートメントを実行することによってほとんどすべてのデータベースへのアクセスを実行します。 これらのステートメントでは、ハードコーディングまたは実行時に、相互運用可能なやデータ ソースに固有で、構築の形式は、アプリケーションのニーズによって異なります。
