# <a name="encryption-access-driver"></a>暗号化 (Access ドライバー)
(このコントロールは、使用可能な場合にのみ Version 4。*x*形式を選択します)。  
  
 Microsoft Jet データベース エンジンの version 4.0 のリリースでは、Access データベースを作成するときにデータベースの暗号化を許可します。 参照してください、 *Jet データベース エンジン プログラマ リファレンス*の追加情報。
