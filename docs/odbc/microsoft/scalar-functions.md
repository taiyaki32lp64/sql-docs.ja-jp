# <a name="scalar-functions"></a>スカラー関数
Microsoft ODBC のデスクトップ データベース ドライバーでは、次のスカラー関数をサポートします。  
  
||||  
|-|-|-|  
|CONCAT|LCASE|[RIGHT]|  
|CONVERT|[LEFT]|[RTRIM]|  
|CURDATE|LENGTH|[SUBSTRING]|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|[LTRIM]|User|  
|DAYOFMONTH|MOD|[YEAR]|  
|[DAYOFWEEK]|[MONTH]||  
  
 引数とスカラー関数の戻り値については、付録 F を参照してください、 *ODBC プログラマ リファレンス。*
