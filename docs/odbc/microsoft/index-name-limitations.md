# <a name="index-name-limitations"></a>インデックス名の制限
Paradox ドライバーを使用する場合、プライマリ インデックスが定義されているテーブルと同じ名前必要があります。 その他のまたは非一意のインデックス定義されたテーブルと同じ名前が必要です。
