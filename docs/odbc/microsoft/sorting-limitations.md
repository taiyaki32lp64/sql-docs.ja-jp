# <a name="sorting-limitations"></a>並べ替えの制限事項
GROUP BY 句、ORDER BY 句、SELECT DISTINCT ステートメント、または外部結合での並べ替えキーの最大長は 255 バイトです。並べ替え行内のすべての並べ替えキーの最大長は、65,500 バイトです。
