# <a name="read-only-status-text-file-driver"></a>読み取り専用の状態 (テキスト ファイル ドライバー)
テキストのドライバーを使用すると、データ ソースのテーブルは読み取り専用です。 ただし、アプリケーションでは、テーブルのテキストの挿入を実行できます。
