# <a name="operators---logical"></a>演算子 - 論理
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  値を評価しでのブール値を返す、データ マイニング拡張機能 (DMX) 式で論理演算子を使用できる[!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]します。  
  
 次の表は、DMX がサポートする論理演算子について示しています。  
  
|演算子|説明|  
|--------------|-----------------|  
|[AND&#40;DMX&#41;](../dmx/and-dmx.md)|2 つの数値式の論理積演算を実行します。|  
|[NOT&#40;DMX&#41;](../dmx/not-dmx.md)|1 つの数値式の論理否定を実行します。|  
|[OR&#40;DMX&#41;](../dmx/or-dmx.md)|2 つの数値式の論理和を実行します。|
